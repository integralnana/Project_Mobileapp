# Mobile Application for Finding People to Share Purchases (โมบายแอพลิเคชั่นการแชร์ซื้อสินค้า)

## Description
โครงงานปริญญานิพนธ์นี้นำเสนอแอปพลิเคชั่นแชร์ซื้อสินค้า พัฒนาด้วยภาษา Dart เฟรมเวิร์ค Flutter ใช้งาน Firebase จัดการระบบหลังบ้าน แจ้งเตือน และเก็บข้อมูลผู้ใช้ Google Maps API ค้นหาตำแหน่งผู้ใช้และจุดนัดพบ แอปพลิเคชั่นนี้ช่วยให้ผู้ใช้ประหยัดค่าใช้จ่าย  และเข้าถึงสินค้าหลากหลาย ผู้ใช้สามารถแบ่งซื้อ แบ่งจ่าย แบ่งปันสินค้า และรับสินค้าได้สะดวกหากอยู่ใกล้เคียงกัน ระบบออกแบบมาเพื่อตอบสนองความต้องการของผู้ใช้ เพื่อประหยัดค่าใช้จ่าย โดยให้แอปพลิเคชันเป็นตัวกลางการแชร์ซื้อสินค้าให้ กับมีการติดต่อระหว่างผู้ใช้บริการด้วยกัน และมีผู้ดูแลระบบคอยดูแลและจัดการ
